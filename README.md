# WellTrack

WellTrack is a lightweight well-being tracking application designed to help users monitor their mood, habits, daily activities, and overall mental wellness.  
The project demonstrates best practices in documentation, transparency, ethical design, and responsible data handling while serving as a functional prototype for well-being monitoring software.

WellTrack aims to support users in improving their mental and emotional health by enabling them to:

- Log daily moods and wellness indicators  
- Track personal habits and routines  
- Visualize progress over time  
- Reflect on patterns that may influence mental well-being  

The project focuses on simplicity, accessibility, and ethical development practices, making it suitable for both personal use and as a learning resource for developers.

## ğŸ” Privacy Notice

WellTrack includes a privacy section to ensure safe and ethical handling of data:
- The current version should not store sensitive personal information
- Any sample data used during development must be anonymized
- Local logs should avoid containing personally identifiable information
- Future production versions will require secure data storage, encryption, and responsible retention policies
- Users running the project locally should avoid long-term storage of personal well-being logs unless protected
- This ensures that even during early development, the project aligns with responsible data practices.
